Banner Images Directory
=======================

This directory should contain the following banner images for the authentication pages:

1. a.png - First banner image (recommended size: 400x250px)
2. b.png - Second banner image (recommended size: 400x250px)
3. c.png - Third banner image (recommended size: 400x250px)

These images will be displayed in a carousel on the authentication pages (login, register, etc.).

If the images are not found, placeholder images will be displayed instead.

Note: You can use any image format supported by web browsers (PNG, JPG, WEBP, etc.), 
but make sure to update the file extensions in the HTML templates if you use different formats.
